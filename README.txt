**何をするスクリプトか？
選択したセル内の文字列が正規表現にマッチしたら、指定のセルスタイルを適用します。

**インストール
スクリプト本体（regex_cellstyle.jsx）を
~/Library/Preferences/Adobe InDesign/Version 6.0-J/ja_JP/Scripts/Scripts Panel/
にコピーしてください。エイリアスを入れておくだけでもかまいません。
スクリプトパレットから使用します。


**使用方法
+「ウインドウ」メニューから「スクリプティング」ー「スクリプト」を選択し、スクリプトパレットを出します。
+検索範囲となるセル（またはテーブル全体）を選択してください。
+スクリプトパレットから、スクリプト「regex_cellstyle.jsx」をダブルクリックします。
+「正規表現でセルスタイル適用」ダイアログが出ます。検索正規表現を入力し、セルスタイルを選んでください。「OK」ボタンをクリックします。
+正規表現にマッチしたセルのみセルスタイルが適用されたのを確認してください。
